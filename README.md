# React学習用クイズアプリケーション

## 仕様

### ページ一覧

1. Topページ
2. クイズページ
3. 結果ページ

### 機能一覧

1. Topページ
    - ボタン:押したらクイズページへと移行
2. クイズページ

   - 問題文を表示
   - 選択肢を表示  
   - 押したら…
   1. 正誤判定を行い記録する
   2-1. 問題文がまだあれば次の問題を表示
   2-2. 問題文がもうなければ結果ページへ遷移

3. 結果ページ
    - 結果までの演出を1~2秒表示させて非表示
    - クラッカー演出
    - 問題総数と正解数を取得し表示
